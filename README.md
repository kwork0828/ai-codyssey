# 과제1 - 대화 로그 

## 재현성 기록
- 채택 모델: Claude Opus 4.8 (네이토 플랫폼)
- 모드: 학습 튜터 모드
- 사용 날짜: 2026-07-__
- 개인정보: 실명·학번 마스킹 처리

## 로그 포인트
- 턴 3~4: 정보 부족 → 되묻기 발동 (환각 방지 확인)
- 턴 5: 조건 변경 (문체 간결화) → 맥락 유지하며 바뀐 부분만 반영

---

```
[턴 1] AI
상담일지 작성을 위해 아래 정보를 입력해 주세요.
- 상담 대화 내용 또는 상담 메모
- 상담 유형
- 상담 대상자 이름 또는 주제

[턴 2] 나
14기 김OO, 화상, 자소서컨설팅 / AI 자소서 초안 사용 /
서류 반복 탈락 스트레스 / 경험 정량화·직무 맞춤 안내

[턴 2] AI
상담일지 3. 14기 김OO
1) 상담신청내용: 이력서/자소서 코칭
2) 현황: 자기소개서 작성 시 AI로 초안을 작성해 활용해 온 편이며,
   서류 전형에서 반복적으로 탈락하며 심리적 부담을 느끼고 있는 상황임.
   이에 자기소개서의 완성도를 높이고 직무 적합성을 강화하기 위해 상담을 신청함.
3) 상담진행내용: 경험을 나열하기보다 성과와 역할을 수치로 제시하는 정량화
   방식으로 정리하도록 안내함. 지원 직무 역량과 본인 경험의 연결이 드러나도록
   직무 맞춤형으로 재구성하도록 피드백함. AI 초안을 그대로 활용하기보다
   본인의 실제 경험과 지원 동기가 담기도록 보완할 필요가 있다고 정리하였음.
   반복 탈락 부담에 대해서는 작성 방향을 점검하며 자신감 회복을 격려함.
4) 기타: 본 상담은 화상으로 진행되었음. 수정 후 재점검이 필요할 경우
   추가 상담을 안내함.

[턴 3] 나
16기 이OO, 대면, 면접코칭 / 모의면접 진행 / 답변 두괄식 부족 /
결론 먼저 말하도록 피드백

[턴 3] AI
상담일지 4. 16기 이OO
1) 상담신청내용: 면접코칭
2) 현황: 면접 준비를 위해 모의면접을 진행하였으며, 결론을 먼저 제시하지
   못하고 부연 설명이 앞서는 경향이 확인됨. 답변 구조 개선을 위해 상담을 신청함.
3) 상담진행내용: 두괄식 구조가 부족하여 핵심 메시지 전달이 약하다고 피드백함.
   결론을 먼저 말한 뒤 근거와 사례를 덧붙이는 방식으로 답변을 구성하도록 코칭함.
   답변이 장황해지지 않도록 핵심을 먼저 정리하는 연습이 필요하다고 안내함.
   전달력과 설득력을 강화하도록 방향을 정리하였음.
4) 기타: 본 상담은 대면으로 진행되었음. 반복 연습 후 필요 시 추가 모의면접을
   안내함.

[턴 4] 나
14기 학생 상담함

[턴 4] AI  ← ★정보 부족, 되묻기 발동
상담일지 작성을 위해 아래 내용을 확인하고자 합니다.
- 상담 대상자 이름(또는 "OO 학생" 표기 여부)
- 상담 유형 (예: 파이널컨설팅, 면접코칭, 자소서컨설팅 등)
- 상담에서 다룬 핵심 내용 또는 대화 메모
위 정보를 입력해 주시면 이어서 상담일지를 작성하겠습니다.

[턴 5] 나
- 홍길동 학생
- 면접코칭
- 최종면접에서 자꾸 탈락하는 이유에 대해 여러 방면으로 함께 고찰해보았음

[턴 5] AI
상담일지 5. 14기 홍길동
1) 상담신청내용: 면접코칭
2) 현황: 최종면접 단계에서 반복적으로 탈락하는 상황으로, 원인을 파악하고
   개선 방향을 모색하기 위해 상담을 신청함.
3) 상담진행내용: 최종면접 반복 탈락 원인을 여러 방면에서 함께 고찰함.
   답변 내용, 전달 방식, 직무 적합성 등 다양한 측면을 점검하며 개선이 필요한
   부분을 짚어보았음. 향후 보완 방향을 함께 정리하였음.
4) 기타: 진행 방식(대면/화상) 확인이 필요함. 도출된 개선 방향을 반복 연습한
   뒤, 필요 시 추가 상담을 안내함.

[턴 6] 나  ← ★조건 변경 (문체 간결화)
문체를 좀 더 간결하게 해줘

[턴 6] AI  ← 맥락 유지하며 바뀐 부분(문체)만 반영
상담일지 5. 14기 홍길동
1) 상담신청내용: 면접코칭
2) 현황: 최종면접에서 반복 탈락 중이며, 원인 파악과 개선을 위해 상담을 신청함.
3) 상담진행내용: 최종면접 탈락 원인을 함께 고찰함. 답변 내용, 전달 방식,
   직무 적합성 등 여러 측면을 점검하고, 보완이 필요한 부분을 짚음. 이를 토대로
   향후 개선 방향을 정리함.
4) 기타: 진행 방식(대면/화상) 확인 필요. 개선 방향을 반복 연습한 뒤,
   필요 시 추가 상담으로 점검하기로 안내함.
```

## 로그 분석 (제출용 요약)
- **되묻기 동작 확인**: 턴 4에서 정보가 부족하자 임의로 작성하지 않고 3개 항목을 되물음 → 환각 방지 정상 작동.
- **조건 변경 대응 확인**: 턴 6에서 "문체 간결화" 요청 시, 상담 내용·구조는 그대로 유지한 채 문장만 축약 → 맥락 유지 능력 확인.
- **상담유형 매핑 확인**: 턴 2에서 입력 "자소서컨설팅"을 목록의 정식 명칭 "이력서/자소서 코칭"으로 정확히 매핑.
